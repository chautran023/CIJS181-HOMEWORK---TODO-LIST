Câu 1: 'var' khai báo một biến có phạm vi trong block?

1. True
2. False
Trả lời: False. Phạm vi global + hoisting. (Hoisting: Di chuyển khai báo biến và hàm lên đầu scope trước khi thực thi)

Câu 2: 'const' khai báo một biến có phạm vi trong block?

1. True
2. False
Trả lời: True. Ko hoisting. Ko gán lại.

Câu 3: 'let' khai báo một biến có phạm vi trong block?

1. True
2. False
Trả lời: True. Ko hoisting. Có thể gán lại

Câu 4: Giá trị sau cùng của 'obj' trong ví dụ sau là gì?

const obj = { foo: 1 }
obj.bar = 2
Trả lời: Đáp án 3

Câu 5: Kết quả của ví dụ sau đây là gì?

const speed = 'quick';
`The ${speed} brown fox jumps over the lazy dog.`
Trả lời: 'The quick brown fox jumps over the lazy dog.' (nếu đề bài có thêm ; sau dòng đầu tiên)

Câu 7 (spread operator): Giá trị của array trong ví dụ sau:

const array = [1, 2, 3]
const extension = [4, 5, 6]

array = [...extension, ...array]
Trả lời: error. (Nếu đề bài có thêm const trước array để khai báo thì kết quả ra: [4, 5, 6, 1, 2, 3])

Câu 8 (spread operator): Giá trị của a trong ví dụ sau?

function mystery(...params) {
return params
};
let a = mystery(1, 23, 4);
Trả lời: [1, 23, 4]

Câu 9 (spread operator): Giá trị của biểu thức sau?

[...[..."..."]].length
Trả lời: 3 (mỗi dấu "." là tính 1 đơn vị, ở đây có 3 dấu ".")

Câu 10: Giá trị của a và b với ví dụ sau?

let a = 12, b = 3;
[a, b] = [b, a];
Trả lời: [3, 12]
